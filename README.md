# nav-nangong-hub

这是一个用于归档和展示多个独立 HTML 页面的仓库。  
仓库本身不针对任何特定域名或网站，仅作为页面文件的集中存放与版本管理用途。

## 项目简介

本仓库主要用于收集、整理和发布若干独立开发的 HTML 页面。  
每个页面可视为一个独立的功能模块或静态展示单元，适合用于学习、参考或快速部署。

## 目录说明

```
nav-nangong-hub/
├── index.html          # 入口页面（可选）
├── pages/              # 独立 HTML 页面存放目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：存放所有独立 HTML 页面，每个页面自成一体，不依赖仓库外资源。
- `assets/`：存放页面所需的样式、脚本、图片等资源文件。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可直接在浏览器中打开使用。
- 页面之间无强关联，可根据需要单独引用或修改。
- 归档内容可能随时间更新或补充，不保证所有页面长期可用。
